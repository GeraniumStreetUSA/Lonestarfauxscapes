What changes the number

Size and height

Longer runs and taller panels use more material and often need more structure.

Mounting surface

A clean wall, an existing fence, masonry, or a custom frame all install differently.

Custom fabrication

Corners, stepped lines, hidden seams, trims, and logo work add shop time.

Fire-rated and commercial requirements

NFPA options, submittals, and coordination can materially change the scope.
